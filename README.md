elementary-stats
================

elementary os stats from Launchpad

#### Configuration needed

* PHP >= 5.x
* Python >= 2.7.x
    * urllib
    * os
    * time
    * sys
    * shutil

===

#### Installation

Copy/paste sources on your server and let it roll !
